TabGrouper Test Page

This page is used to test the TabGrouper Chrome extension.

Try pressing Cmd+H (Mac) or Ctrl+H (Windows/Linux) to open the TabGrouper search box.

Test Instructions:

  1. Install the TabGrouper extension in Chrome
  2. Open this page in Chrome
  3. Press the keyboard shortcut: Cmd+H or Ctrl+H
  4. The TabGrouper search interface should appear

Debugging:

If the shortcut doesn't work:

  • Check Chrome's Developer Tools Console for errors
  • Verify the extension is loaded in chrome://extensions/
  • Make sure the shortcut is configured in Chrome's extension shortcuts
  • Try refreshing this page after loading the extension
